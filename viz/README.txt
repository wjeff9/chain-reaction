DESCRIPTION
Our visualization is a dashboard built with D3 and React, tailored to explore e-commerce connections and delivery patterns. It features three innovative and intuitive components: a Map, a Delivery Timeline, and a Visual Model Representation. The system relies on a drill-down approach to balance aggregation and granularity, allowing users to adjust the visualization to suit their desired level of detail.

The primary component is the Map, which illustrates the primary innovation offered by our Graph Neural Network (GNN) model: associating orders related by location and time. To prevent visual clustering, it uses a three-level hierarchy (location, customer/seller, and order). It includes a weekly time range slider and allows filtering by customer/seller view and delivery outcomes (on time/late). The second component is the Delivery Timeline, a Gantt Chart that splits orders into phases to help stakeholders quickly identify bottlenecks leading to delays. Finally, the Visual Model Representation illustrates how our predictive model works under the hood by plotting an order and its k-nearest neighbors (k=5) based on customer location and temporal proximity.

INSTALLATION
1. Ensure you have Node.js and npm installed on your system.
2. Open a terminal and navigate to the "viz" directory
   $ cd viz
3. Install the required JavaScript dependencies (React, D3, Vite, etc.) by running:
   $ npm install

EXECUTION
1. After the installation is complete, start the local development server by running:
   $ npm run dev
2. In your terminal, you will see a local URL generated by Vite (typically http://localhost:5173/). 
3. Open your preferred web browser and navigate to that URL to interact with the dashboard.
4. Alternatively, you can build and preview the production-ready application by running:
   $ npm run build
   $ npm run preview
